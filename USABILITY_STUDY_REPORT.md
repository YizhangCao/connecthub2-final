## Tasks

Participants were asked to complete the following tasks:

| Task # | Task Description | Success Criteria |
|--------|------------------|------------------|
| 1 | Find a specific contact by name | Contact details displayed |
| 2 | Add a new contact with all fields | Contact appears in list |
| 3 | Log an interaction for a contact | Interaction appears in timeline |
| 4 | Identify contacts needing follow-up | Dashboard stat acknowledged |
| 5 | Delete a contact | Contact removed from list |
